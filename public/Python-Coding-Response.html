To solve the problem of finding all starting indices of substrings in s that are a concatenation of each word in words exactly once, we need to consider several factors:

  1. Understanding the Problem:

    • We are given a string s and a list of words words.
    • Each word in words is of the same length.
    • We need to find all starting indices of substrings in s that are a concatenation of each word in words exactly once and without any intervening characters.

  2. Constraints and Requirements:

    • The solution must efficiently handle the constraints: 1 s.length 10^4, 1 words.length 5000, and 1 words[i].length 30.
    • The solution should return the indices in any order.

  3. Approach:

    • Calculate the total length of the concatenated string, which is total_length = len(words) * len(words[0]).
    • Use a sliding window approach to check substrings of this length in s.
    • For each starting point in the string s, check if it can form a valid substring using all words exactly once.

  4. Algorithm Design:

    • Use a dictionary to count occurrences of each word in words.
    • For each possible starting index i in s, extract the substring of length equal to total_length.
    • Check if this substring can be divided into valid words present in the words list using a dictionary to count occurrences.
    • If it matches, record the starting index.

Here's the implementation of the solution:

class Solution(object):
    def findSubstring(self, s, words):
        """
        :type s: str
        :type words: List[str]
        :rtype: List[int]
        """
        from collections import Counter
        
        if not s or not words:
            return []
        
        word_length = len(words[0])
        total_length = len(words) * word_length
        word_count = Counter(words)
        
        result_indices = []
        
        # Iterate through the string with a window size equal to total_length
        for i in range(len(s) - total_length + 1):
            seen_words = {}
            # Check every word-length substring within this window
            for j in range(len(words)):
                word_start = i + j * word_length
                word = s[word_start:word_start + word_length]
                
                if word in word_count:
                    seen_words[word] = seen_words.get(word, 0) + 1
                    # If a word appears more times than it appears in words, break
                    if seen_words[word] > word_count[word]:
                        break
                else:
                    break
            else:
                # If we didn't break from the loop, it means we found a valid start
                result_indices.append(i)
        
        return result_indices

Explanation

  • Sliding Window: The algorithm uses a sliding window to examine each potential starting index for a valid concatenated substring. It iterates over all possible starting points that allow for a full concatenation of the words.

  • Word Counting: We utilize two hash maps (dictionaries): one (word_count) to store the count of each word in words, and another (seen_words) to keep track of how many times we've seen each word in the current window.

  • Efficiency: By limiting checks to only possible valid windows and using hash maps for quick lookups, we ensure that our approach is efficient enough to handle the constraints provided.
